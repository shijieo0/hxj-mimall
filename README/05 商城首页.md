### base.scss 抽取公共固定样式模块

1.在base.scss中抽取公共固定样式代码

```
// 项目开发过程中，定义一些公共样式，在需要的vue文件样式区头部导入即可生效
.container {
    width: 1226px;
    margin-right: auto;
    margin-left: auto;
}
```

2.在vue文件中样式区头部导入即生效

### mixin.scss 抽取公共样式函数用法

#### 1.在mixin.scss文件中定义公共样式函数

```
// flex布局样式复用函数
@mixin flex($hov:space-between, $col:center) {
    display: flex;
    justify-content: $hov;  // 元素在水平方向如何安放，默认水平居中
    align-items: $col;      // 元素在垂直方向如何安放，默认垂直居中
}
// 背景图片样式复用函数
@mixin bgImg($w:0,$h:0,$img:'',$size:contain) {
    display: inline-block;
    width: $w;
    height: $h;
    background: url($img) no-repeat center;
    background-size: contain; // 让把背景图充满元素
}
```

#### 2.在vue文件中样式区头部导入

```
<style lang="scss">
@import './../assets/scss/mixin.scss';
...
</style>
```

#### 3.在样式中引用

```
.container {
            @include flex();    // 复用flex布局样式函数，使用默认值
            a {
                display: inline-block;
                color: #B0B0B0;
                margin-right: 17px;
            }
            .my-cart {
                width: 110px;
                background-color: #FF6600;
                text-align: center;
                color: #ffffff;
                .icon-cart {
                	// 引用背景图样式函数，按实际情况传值即可
                    @include bgImg(16px,12px,'/imgs/icon-cart-checked.png',contain);
                    margin-right: 4px;
                }
            }
        }
```
